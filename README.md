# Dynamic Load JS
Lightweight, Scalable

## Installation
Hosted by github
```html
<script type="text/javascript" src="https://tpnet3.github.io/dynamic-load-js/dist/latest/DynamicLoad.min.js"></script>
```

Download via bower
```
bower install dynamic-load-js
```
